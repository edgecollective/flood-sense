## Wiring

Feather M0 board with Ultrasonic Sensor, LoRa Breakout and TPL5110 Timer

### LoRa Breakout -- Feather M0
CS -- A3
RST -- A4
INT -- A5

## Ultrasonic -- Feather M0
TRIG -- 10
ECHO -- 11 (via voltage divider)

### Timer -- Feather M0
DRV -- BAT
DONE -- 12
